# 客户池页面设计规范 (v2)

> 基于客户池 UI 截图提取的设计规范，适用于 B 端管理后台系统。

---

## 1. 颜色系统 (Color System)

### 品牌色 / 功能色
| 用途 | 色值 | 说明 |
|------|------|------|
| 主操作色 | `#1890FF` | 查询按钮、选中态、操作链接、图标高亮 |
| 成功/已分配状态 | `#13C2C2` | 状态标签文字 |
| 危险/未分配状态 | `#F5222D` | 状态标签文字 |

### 文字颜色
| 层级 | 色值 | 用途 |
|------|------|------|
| 主文字 | `#333333` / `#262626` | 客户名称、手机号、重要信息 |
| 次要文字 | `#666666` / `#8C8C8C` | 表单标签、面包屑、描述文字 |
| 占位符文字 | `#BFBFBF` | 输入框 placeholder |

### 背景色
| 区域 | 色值 | 说明 |
|------|------|------|
| 页面底色 | `#F0F2F5` | 整体页面背景 |
| 容器背景 | `#FFFFFF` | 卡片/面板背景 |
| 表格表头 | `#FAFAFA` | 极淡灰色，表头背景 |

### 边框 / 分割色
| 用途 | 色值 |
|------|------|
| 输入框/按钮边框 | `#D9D9D9` |
| 表格行分割线 | `#F0F0F0` |

### 状态标签颜色
| 状态 | 背景色 | 文字色 |
|------|--------|--------|
| 已分配 | `#E6FFFB` | `#13C2C2` |
| 未分配 | `#FFF1F0` | `#F5222D` |

---

## 2. 字体规范 (Typography)

### 字体家族
```
PingFang SC, Microsoft YaHei, -apple-system, sans-serif
```

### 字号 & 字重
| 元素 | 字号 | 字重 |
|------|------|------|
| 页面标题（客户列表） | `16px` | Bold (600) |
| 表单标签 / 表格内容 | `14px` | Regular (400) |
| 操作按钮 / 面包屑 | `14px` 或 `12px` | Regular (400) |
| 表头文字 | `14px` | Medium/Bold (500/600) |

---

## 3. 间距规范 (Spacing)

### 页面级
- **页面外边距**: 左右及上下约 `24px`

### 容器内边距 (Padding)
- **搜索区域 / 列表区域**: `24px`
- **表格单元格**: 垂直 `12px - 16px`，水平 `16px`

### 元素间距
- **搜索项之间水平间距**: `24px` 或 `32px`
- **Label 与 Input 之间**: `8px`
- **按钮组之间**: `8px` 或 `12px`

---

## 4. 组件样式 (Component Styles)

### 按钮 (Buttons)

| 按钮类型 | 背景色 | 文字色 | 边框 | 圆角 |
|----------|--------|--------|------|------|
| 主按钮（查询） | `#1890FF` | `#FFFFFF` | 无 | `2px - 4px` |
| 次级按钮（重置） | `#FFFFFF` | `#666666` | `1px solid #D9D9D9` | `2px - 4px` |
| 幽灵按钮（添加/导入/导出） | 透明 | `#1890FF` | 无 | `2px - 4px` |

### 输入框 / 选择框 (Inputs / Selects)
- **高度**: 标准 `32px`
- **边框**: `1px solid #D9D9D9`
- **圆角**: `2px`
- **Focus 状态**: 边框变为 `#1890FF`（蓝色）

### 表格 (Table)
- **表头**: 背景 `#FAFAFA`，文字加粗，底部 `1px` 分割线
- **行样式**: 每行底部 `#F0F0F0` 分割线
- **Hover 状态**: 浅蓝色背景变化
- **选择框**: 正方形，勾选时填充主色蓝 `#1890FF` 并带白色对勾

### 标签 (Tags / Status Badges)
- **高度**: 约 `22px`
- **字号**: 约 `12px`
- **圆角**: 通常为 `2px` 或小圆角
- **样式**: 带背景色的轻量化标签

---

## 5. 布局结构 (Layout Structure)

### 整体布局
- **经典后台布局**: 顶部导航 + 左侧菜单（收起状态）+ 右侧内容区

### 面包屑
- 位于页面最上方，显示当前路径导航

### 搜索区域 (Filter Area)
- 位于内容区顶部
- 采用栅格布局（通常 4 列）
- 底部对齐放置操作按钮（重置、查询）
- 表单项包含 Label + Input/Select 左右排列

### 操作栏
- 左侧：页面标题 "客户列表"
- 右侧：功能按钮组（添加、导入、导出）

### 数据表格
- 占据页面核心位置
- 横向撑满容器
- 支持勾选操作

---

## 6. 其他设计细节

### 图标风格
- **类型**: 线性（Outlined）风格
- **线宽**: 约 `1px - 1.5px`，粗细均匀
- **颜色**: 继承文字颜色或 `#666666`

### 阴影
- 容器组件（白色卡片）通常带有极细微的投影
  - Blur: `8-10px`
  - Opacity: `0.05` 左右
- 或仅靠背景色差区分

### 操作列
- 高频操作（查看、编辑、领取客户等）以文字链接形式呈现
- 中间有灰色垂直分割线分隔

---

## 设计特征总结

> 该界面体现了 **高效、清晰、专业** 的 B 端产品特征，参考了 Ant Design / Element UI 的设计语言：
> - 注重信息展示效率和视觉规整性
> - 栅格化布局，间距统一
> - 状态色彩区分清晰
> - 交互反馈直观
