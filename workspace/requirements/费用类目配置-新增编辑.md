| 序号 | 字段名称 | 英文ID | 类型 | 字段长度 | 是否必填 | 是否编辑 | 默认值 | 说明 |

## 4. 交互说明

### 4.1 按钮逻辑

| 场景 | 编辑按钮 | 取消按钮 | 提交按钮 |
|------|----------|----------|----------|
| 新增页（点击【新增】进入） | 隐藏 | 显示 | 显示 |
| 编辑页（点击【编辑】进入） | 显示 | 显示 | 显示 |
| 查看页（点击【查看】进入） | 显示 | 隐藏 | 隐藏 |

### 4.2 按钮行为

- **【编辑】**：点击后，详情页字段变为可编辑状态
- **【取消】**：关闭详情页，返回列表页
- **【提交】**：
  - 从【新增】进入 → 关闭详情页，返回列表页，插入一行新数据
  - 从【编辑】进入 → 关闭详情页，返回列表页，更新对应行数据

### 4.3 费用类目结构（树形）

- 支持展示多级树形结构
- 每个节点右侧有操作图标（展开/折叠/编辑）
- 底部有【+ 添加一级目录】按钮
- 子节点右侧有【+ 添加】按钮

## 5. 字段组件类型

| 字段 | 组件类型 | 说明 |

## 6. 用户故事

### 6.1 新增费用类目
> AS A 财务管理员  
> I WANT 在费用类目配置页新增一个费用类目  
> SO THAT 对费用进行分类管理

**验收标准：**
- [ ] 点击【新增】按钮，弹出新增编辑弹窗
- [ ] 上级目录自动带入当前选中节点（若无选中则为顶级）
- [ ] 费用类目编码自动生成，不可编辑
- [ ] 必填字段未填写时，提交按钮禁用或提示
- [ ] 提交后关闭弹窗，列表刷新显示新数据

### 6.2 编辑费用类目
> AS A 财务管理员  
> I WANT 编辑已有的费用类目信息  
> SO THAT 修改费用类目配置

**验收标准：**
- [ ] 点击列表【编辑】按钮，进入编辑弹窗并回显数据
- [ ] 费用类目编码不可修改
- [ ] 修改后提交，数据更新并刷新列表

### 6.3 查看费用类目
> AS A 财务管理员  
> I WANT 查看费用类目的详细信息  
> SO THAT 确认类目配置内容

**验收标准：**
- [ ] 点击列表【查看】按钮，进入查看模式
- [ ] 所有字段只读显示
- [ ] 取消/提交按钮隐藏，编辑按钮显示
